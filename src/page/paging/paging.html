后端分页控件

简介

后端查询量比较大的页面需要改造为后端分页，大体需要有三个地方改造一下。后面的两个部分，算是一劳永逸的复制一次，以后拷贝就行了

  • 1.后端提供数据的process类方法改造
  • 2.拷贝分页PageModelAndview类到项目中
  • 3.拷贝分页html的jsp文件到项目中

1.后端提供数据的process类方法改造

提供数据的process类方法，必须支持两个条件参数的查询，1：开始位置，2：查询长度

                    
        int psize=100;// 一页数据长度
        int p = Utils.parseInteger(request.getParameter("p"), 1);// 从参数中获取分页
        int skip = (p - 1) * psize;// 计算起始位置
        long s = Utils.parseLong(request.getParameter("s"), System.currentTimeMillis() - 1 * 86400000);// 查询开始时间
        long e = Utils.parseLong(request.getParameter("e"), System.currentTimeMillis());// 查询结束时间

        long count = MyBeanProcessor.getCount(s,e,null);//获取查询条件下的数据长度
        List<MyBean> list = MyBeanProcessor.getMyBeanList(s,e skip, p);// 获取查询条件下的数据
        // 使用 分页的modelandview,代码在下一小节，复制拷贝即可
        ModelAndView mv = new PageModelAndview("admin/xxx", p, count, psize);
        mv.addObject("list", list);// 添加数据

2.PageModelAndview类

项目中新建一个 PageModelAndview 类，然后将下面代码拷贝过去即可（此类可以放在基础类文件夹下，方便管理）

                    
        package com.yuncheng.zyuser.modelandview;

        import java.io.IOException;
        import org.springframework.web.servlet.ModelAndView;

        /**
         *
         * @desc 分页modelandview
         * @param p 当前分页
         * @param allsize 总条数
         * @param pagesize 单页长度
         */
        public class PageModelAndview extends ModelAndView {

            public PageModelAndview(String view, int p, long allsize, long pagesize) throws IOException {

                super(view);

                long showpage = 5;
                long begin = 1;// 起始页面
                long allpage = (allsize / pagesize);//总页数
                long end = 1; // 结束页面
                boolean before = false;
                boolean next = false;
                boolean haspage = true;
                //计算起始页,结束页
                if (allpage <= showpage) {
                    begin = 1;
                    end = allpage;
                } else if (p <= (showpage / 2 + 1)) {
                    begin = 1;
                    end = showpage;
                    next = true;
                } else if (p <= (allpage - (showpage / 2))) {
                    before = true;
                    next = true;
                    begin = p - (showpage / 2);
                    end = p + (showpage / 2);
                } else {
                    before = true;
                    begin = allpage - showpage + 1;
                    end = allpage;
                }
                if (p != 1 && allpage > 1) {
                    before = true;
                }
                if (allpage > 1 && p != allpage) {
                    next = true;
                }
                if (allpage < p) {
                    haspage = false;
                }
                super.addObject("haspage", haspage);
                super.addObject("cpage", p);
                super.addObject("pageBegin", begin);
                super.addObject("pageEnd", end);
                super.addObject("pageNext", next);
                super.addObject("pageBefore", before);
            }
        }

3. 分页的jsp模块

分页需要html,js,css 这些模块必不可少，但这些东西不能整合到后端js库中，因为不是每个页面都需要分页，并且分页这种模块理应该属于后端业务页面的一部分,而不是存在于公共库中

在项目中新建paging.jsp （最好放/admin/page,或者其他已经存在的公共组件文件夹下）将下面代码拷贝过去即可

                    
        <%@page contentType="text/html" pageEncoding="UTF-8"%>
        <%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
        <c:if test="${haspage&& pageEnd>1}">
            <style type="text/css">
                .page-op{width:320px;float:right}
                .page-op .page-btn{float:left;width:78px;border:1px solid #dfdfdf;background:#fff;font-size:12px;height:24px;line-height:24px;text-align:center;cursor:pointer}
                .page-op .page-btn:hover{border-color:#b8c3e0}
                .page-op .page-link{background: #fff;display:block;float:left;width:24px;height:24px;border-radius:3px;font-size:12px;border:1px solid #dfdfdf;text-align:center;line-height:24px;cursor:pointer;margin:0px 4px}
                .page-op .page-link.current{border:none;background:#f61d64;color:#fff}
                .page-op .page-link:hover{border-color:#b8c3e0}
                .page-wrap.fix{
                    position:fixed;
                    right:0px;
                    bottom: 0px;
                    width: 360px;
                    padding-top: 10px;
                    height: 40px;
                    background: rgba(0,0,0,0.3);
                }
            </style>
            <div class="page-wrap fix">
                <div class="page-op clear fix" id='page-op'  ${pageBegin} ${pageEnd}>
                    <c:if test="${pageBefore}">
                        <a class="pre-page page-btn"  p="${cpage-1}" >上一页</a>
                    </c:if>
                    <c:forEach var="item" varStatus="c" begin="${pageBegin}" end="${pageEnd}">
                        <a class="page-link  <c:if test="${c.index==cpage}">current</c:if>"  p="${c.index}" >${c.index}</a>
                    </c:forEach>
                    <c:if test="${pageNext}">
                        <a class="next-page page-btn" p="${cpage+1}"  >下一页</a>
                    </c:if>
                </div>
            </div>
            <script type="text/javascript">
                $(function () {
                    $("#page-op .page-btn,#page-op .page-link").on("click", function () {
                        if (!$(this).hasClass("current")) {
                            var p = $(this).attr("p");
                            window.location.search = Ut.search('p', p);
                        }
                    })
                })
            </script>
        </c:if>

1.最后一步，分页业务页面

业务页面只需要做两件事

  • 1. table 定制化
  • 2. 引入分页组件
                
        <table class="table table-bordered" id="zytable" hidepage="true">
        // id 需要设置为 zytable
        // hidepage="true" 隐藏掉datatable 的分页组件
        </table>
        <%--引入分页组件（必须）--%>      
        <jsp:include page="../page/paging.jsp"></jsp:include>

最后的提醒

这个适用于章鱼后端页面， 页面中需要有引用章鱼后端框架，其他页面尚未测试
